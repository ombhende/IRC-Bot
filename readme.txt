Well, Hello Friends and Welcome to my First ever project made based on PHP!! Yeah it's an IRC Bot Written in PHP using Sockets

Version: v1.0.0
_______________
Installing:

Linux: (Use Following Commands)
wget https://github.com/ombhende/IRC-Bot/archive/master.zip
unzip master.zip (If you don't have unzip, type apt-get install zip)
nano bot.php (Edit Config)
php bot.php (Runs your bot for Temp Time)
screen -dmS bot php bot.php (Runs your bot also after you close your VPS
